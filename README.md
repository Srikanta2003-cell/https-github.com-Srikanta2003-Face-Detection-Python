# Face Detection Using Python

## Overview

This project is a real-time **Face Detection System** developed using **Python** and **OpenCV**. It detects human faces from a webcam feed and highlights them with a bounding box.

## Features

* Real-time face detection using webcam
* Detects multiple faces simultaneously
* Fast and lightweight implementation
* Easy to run on any Python environment

## Technologies Used

* Python
* OpenCV
* NumPy

## How to Run

1. Clone the repository
2. Install the required libraries:

   ```bash
   pip install opencv-python numpy
   ```
3. Run the program:

   ```bash
   python face_detection.py
   ```

## Project Outcome

Successfully implemented a real-time face detection application that captures video from a webcam, processes frames using OpenCV, and detects faces with good accuracy and performance.
